# React App

A small react application built with React, Typescript and bundled with Vite

----
## Install dependecies

```bash
  yarn
```
or
```bash
  npm install
```

----
## Run the app locally
```bash
  yarn dev
```
or
```bash
  npm run dev
```

----
## Run production build
```bash
  yarn build && yarn preview
```
or
```bash
  npm run build && npm run preview
```
